## 1. Description and Motivation

### Overview

**Parallel Genetic Algorithm** is used effectively to make calculations and perform actions for the real world **Traffic Control Systems**.

### Motivation

Real world traffic gets too worse in especially smart cities in INDIA. **Accidents** when combine with the already existing traffic becomes a nightmare to people. Traffic systems right now are not intelligent enough to detect accidents and route traffic accordingly such that the waiting time of the road users reduce drastically.
